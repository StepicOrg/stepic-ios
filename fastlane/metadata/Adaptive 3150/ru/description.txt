Этот курс состоит из заданий, которые помогут Вам проверить свои знания вокабуляра, требуемого для сдачи TOEFL. Задачи составлены с помощью SkyEng Dictionary API.

Обратите внимание, что система сама будет автоматически подбирать вам вопросы, учитывая уровень знаний. Вы можете ей помогать, отмечая, какие вопросы оказались для вас самыми лёгкими, а какие слишком сложными - достаточно смахнуть карточку с вопросом.
